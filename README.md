# Simple Student Pass/Fail Predictor

## Overview
This project predicts whether a student will pass or fail based on study hours, attendance, and assignment score using machine learning in Python.

## Objective
The aim of this project is to apply basic AI/ML concepts to a simple academic problem.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Joblib

## Files
- student_data.csv
- train_model.py
- predict.py

## How to Run

### Install libraries
pip install pandas scikit-learn joblib

### Train the model
python train_model.py

### Run prediction
python predict.py

## Input Features
- Study hours
- Attendance
- Assignment score

## Output
- Pass
- Fail

## ML Concept Used
This project uses supervised learning and classification.
